Formato de caracteres en HTML

Formatos básicos

HTML es un lenguaje que fue evolucionando con el tiempo para permitir documentos más complejos. Sin embargo, aún se mantienen algunas etiquetas que permiten dar un formato muy básico. Aunque esto es desaconsejado y debe ser reemplazado con CSS, pueden ser útiles en ciertos casos:

  • <b>...</b> define contenido en negritas
  • <i>...</i> define contenido en cursivas
  • <u>...</u> define contenido en subrayado
  • <s>...</s> define contenido en tachado

Otros formatos

  • <sup>...</sup> define un superíndice
  • <sub>...</sub> define un subíndice
  • <cite>...</cite> define una cita
  • <address>...</address>
    define una
    dirección
  • <dt>: Término
    <dd>: Descripción del término

Entidades HTML

Las entidades HTML son códigos especiales para representar caracteres especiales cuando no es posible mostrarlos directamente:

Formato: &código;

Ejemplos útiles:

  • á: &aacute;
  • é: &eacute;
  • í: &iacute;
  • ó: &oacute;
  • ú: &uacute;
  • ñ: &ntilde;
  • Ñ: &Ntilde;
  • <: &lt;
  • >: &gt;
  • ©: &copy;
  • ®: &reg;
  • €: &euro;
  • &: &amp;
  • Espacio: &nbsp;

Formato con fuente, tamaño y color (obsoleto)

La etiqueta <font> permite definir fuente, tamaño y color, pero está en desuso y debe reemplazarse por CSS.

Ejemplo:

<font face="arial" size="2" color="green">Texto verde en Arial</font>
